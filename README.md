# Solder Ninja Webapp

A web-based controller for the [Solder Ninja Pen](https://www.solder.ninja/pen). Connect your device via USB and control temperature settings, customize your device with a personal name and icon, and update firmware directly from your browser.

## Features

- Temperature control with real-time monitoring
- Device customization with user name and custom icon editor
- Product information display
- Firmware update management with automatic version checking

## Usage

Open [the website](https://sitronlabs.github.io/Solder-Ninja-Webapp/) in a modern browser that supports the Web Serial API (Chrome, Edge, or Opera). Connect your Solder Ninja Pen via USB and click "Connect to a USB device" to begin.

## Requirements

- Modern browser with Web Serial API support
- Solder Ninja Pen (product code SLTO00001)
- USB connection
